  • To check if the API is running goto url :
    • http://127.0.0.1:3000/api/version
  • To know the vertex's min or max id using the service goto the url :
    • http://127.0.0.1:3000/api/graph/info
  • You can find the 'fastest' path from vertex's id: 0 to vertex's id: 4 by using this url :
    • http://127.0.0.1:3000/api/graph/findPath?from=0&to=4&by=fast
  • To find the 'eco' path from vertex's id: 4 to vertex's id: 2 by using this url :
    • http://127.0.0.1:3000/api/graph/findPath?from=4&to=2&by=eco

The "by" parameter can be equal to fast or eco.
